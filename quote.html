Business info

Auto-load pricing from repo CSV

Put a file called pricing.csv in the same folder as quote.html in your repo. Edit the CSV in GitHub to update prices — they'll pick up automatically on the next page load. The tool also keeps a fallback set of prices locally in case the CSV ever can't be reached.

Manual / Google Sheets pricing (fallback)

How to set this up: In your Google Sheet → File → Share → Publish to web → "Entire document" → "Web page" → Publish. Then paste the sheet ID (or full URL) above. Your sheet must have three tabs named exactly: Cameras, Travel, Rates. See the setup guide for the exact column layout.
